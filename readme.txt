Create advanced discount rules for WooCommerce with an intuitive admin panel – no coding required.

== Description ==

Create and manage custom discounts in WooCommerce with ease. Discountly allows you to define flexible discount rules, organize discounts by category, and schedule offers without any coding knowledge.

This plugin uses modern JavaScript tools (including React and WordPress Gutenberg packages) and the final production assets are generated using @wordpress/scripts.

To comply with WordPress.org guidelines and ensure full code transparency, the complete uncompressed source code is publicly available in our GitHub repository:

https://github.com/nikanwp/discountly

== Installation ==

1. Upload the plugin files to the /wp-content/plugins/discountly directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. After activation, go to the "Discountly" menu in the WordPress admin sidebar to start creating discounts.

== Build Instructions ==

This plugin is developed using modern front-end tools.

To install and build the assets:

1. Run: npm install
2. Then run: npm run build

To start development with live updates, use:

npm start

Build scripts are defined in package.json using @wordpress/scripts.
